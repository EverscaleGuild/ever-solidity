# Experimental Solidity Compiler Driver

## Build and Install

### Ubuntu Linux / MacOS

```shell
sh ./compiler/scripts/install_deps.sh
cd ./sold
cargo build --release
```

### Windows 10

Install Visual Studio Build Tools, Rust, Git, cmake.
Run Visual Studio Developer Command Prompt

```shell
cd sold
cargo build --release
```

## Usage

Type `sold --help` for details.
